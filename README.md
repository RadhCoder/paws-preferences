A fun web application that lets users discover their favourite cats by swiping left or right, similar to popular dating apps.

## Features
- Swipe right to LIKE
- Swipe left to DISLIKE
- Mobile-friendly interface
- Images sourced from Cataas API
- Summary page showing liked cats
- Restart option

## Technologies Used
- HTML
- CSS
- JavaScript
- Cataas API

## How to Run
Open `index.html` in any modern browser.
